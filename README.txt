To start searching for flights to your favorite holiday destination, first compleate the following steps...

  Step 1. Download files to some folder.
  Step 2. Navigate to flights_website.
  Step 3. Run the following cmds in sucstation 
  
    flights_website>venv\Scripts\activate             // To activate the vertual enviroment
    (venv)flights_website>set FLASK_APP=wsgi.py       // Set the intial file 
    (venv)flights_website>set FLASK_DEBUG=1           // Not nessasary but recommended (No Spaces)
    (venv)flights_website>flask run                   // Run the file 
  
  
 Step 4. upon having this display in the cmd, copy the url(http://127.0.0.1:5000/ ), 
        and paste it into your favorite browser(provided its not Explorer...)
